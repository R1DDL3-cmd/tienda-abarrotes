# Instrucciones de Instalacion - Sistema Tienda de Abarrotes

## Para usuarios sin conocimientos tecnicos

### Paso 1: Requisitos

Necesitas:
- Una PC con Windows 7, 8, 8.1, 10 u 11
- Una tablet Android (opcional, para acceder desde la tablet)
- Un router WiFi (para conectar la PC y la tablet)

### Paso 2: Instalar el programa

1. Abre la carpeta donde tienes el programa
2. Busca la carpeta `dist`
3. Dentro de `dist`, haz doble clic en el archivo **Instalar.bat**
4. Si Windows pregunta "Permitir que esta aplicacion haga cambios en el dispositivo?", haz clic en **Si**
5. Espera a que termine la instalacion (aparecera un mensaje verde)
6. Presiona cualquier tecla para cerrar la ventana

### Paso 3: Iniciar el programa

1. Ve al escritorio de tu PC
2. Haz doble clic en el icono **Tienda Abarrotes** (tiene forma de tienda)
3. Espera unos segundos a que se abra la ventana
4. Veras la pantalla de inicio de sesion

### Paso 4: Iniciar sesion

Usa estas credenciales para entrar:
- **Usuario:** admin
- **Contrasena:** admin123

(El cajero puede usar: cajero / cajero123)

### Paso 5: Conectar la tablet (opcional)

1. En la PC, mira la pantalla de inicio de sesion
2. Busca la linea que dice: "Accede desde la tablet: http://192.168.X.X:3000"
3. Anota esa direccion (ejemplo: http://192.168.1.5:3000)
4. En tu tablet, abre el navegador (Chrome, por ejemplo)
5. Escribe la direccion que anotaste
6. Veras la misma pantalla de login en la tablet

### Paso 6: Usar el sistema

Ya puedes:
- **Vender:** Escanea productos con el lector de codigo de barras o buscalos por nombre
- **Agregar productos:** En el modulo de Inventario
- **Ver reportes:** En el modulo de Contabilidad
- **Hacer respaldos:** En Contabilidad > pestana Respaldos

### Solucion de problemas comunes

**"No encuentro el archivo Instalar.bat"**
- Busca dentro de la carpeta `dist`
- Si no ves la carpeta `dist`, busca un archivo llamado `Instalar.bat` en la carpeta principal

**"La tablet no se conecta"**
- Asegurate de que la tablet y la PC esten conectadas a la MISMA red WiFi
- Verifica la direccion IP en la pantalla de login de la PC
- Si el firewall de Windows bloquea, permite el programa en el firewall

**"Se me olvido la contrasena"**
- Reinstala el programa (esto borrara todos los datos)
- O contacta al desarrollador para recuperar el acceso
